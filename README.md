# Posti Sede

## Setup

```bash
npm install
```

## Environment

```bash
cp .env.sample .env
```

Fill the `.env` file with the correct values.

- `EXPO_PUBLIC_DATABASE_URL`: The URL of the database.
- `EXPO_PUBLIC_DATABASE_DRIVER`: The driver of the database. (neon, postgres, etc.)

> [!NOTE]
> For now only neon is supported.

## Run

```bash
npm start
```
